SpotCommander

The most intuitive and feature-rich remote control for Spotify for Linux

SpotCommander is a remote control for Spotify for Linux, optimized for mobile devices. It works on any device with a modern browser, and it's free and open source.

SpotCommander is the most elegant, intuitive, feature-rich and universal remote control for Spotify, exclusive for Linux users!

Yes, it works without a Premium subscription!

If you, like me, have a Linux-powered HTPC connected to a good sound system, being able to control Spotify wherever you are, using whatever device you've got, is great.

The Android app alone has been downloaded more than 50,000 times, and SpotCommander is used by lots of people all over the world, every day.

Advantages over Spotify Connect

  • Works without a Premium subscription
  • Control both Spotify's volume and the system volume
  • Suspend and shut down the computer
  • Queue with additional actions, so you can pause, stop, suspend or shut down after tracks in the queue
  • Lyrics
  • Works on any device with a modern browser
  • Many users can control the same Spotify client without being logged in as the same user
  • Android app with several extra features (see below)
  • Android app widget with more features
  • Links to Last.fm, Wikipedia and YouTube
  • Various other features
